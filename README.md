# Cybersecurity Toolkit

A comprehensive Python-based security toolkit for network analysis, vulnerability assessment, and password management.

## Features

- **Port Scanner**: Scan target systems for open ports with banner grabbing
- **Network Scanner**: Discover active devices on local network using ARP
- **Password Manager**: Securely store passwords with Fernet encryption
- **Vulnerability Scanner**: Detect SQL injection and XSS vulnerabilities
- **File Integrity Checker**: Monitor files for unauthorized changes
- **WHOIS Lookup**: Get domain registration information

## Requirements

- Python 3.8 or higher
- Administrator/root privileges for network scanning
- Internet connection for WHOIS lookups
